通往 **[[阿西穆德·诺克 Ahmed Noke]]** 的法师塔的小路在城镇外的主路几英里之外分 出，在灌木丛的空隙间突然出现。这条路铺得很好，在高大 的橡树之间穿梭着延伸而去。 

任何检查道路的人都会轻易认出地面上留下的许多脚印 与爪印，其中最新的那组看起来正对得上 **[[古兹 Guz]]** 那群家伙的脚 印。 

这条小径在稀疏的林地中继续延伸了大约一英里，除非 队伍的成员们已经知道前方等待的是什么，不然在目睹 **[[芬塞尔·闪布莱特 Finethir Shinebright]]** 的旧屋自树顶探出时他们所看到的景象将使他们感到十 分意外。

	你面前的房子不是石头或玻璃，而是由四棵坚固的 橡树的活树枝组成的。它们经过成型和编织，形成三个厚平台。 
	
	这些平台中最低的一个宽约 40 英尺，距地面约 10 英尺。唯一明显的上行路线是一条由树根和树枝组成的 缓坡，与主路大致相连。 
	
	树枝卷曲在它的基部周围，在它周围形成一个粗糙 的碗形状。从你站的地方可以看到花朵和小树生长在它 的边缘。 
	
	到目前为止，三个平台中最大的是中间平台，看起 来大约有 60 英尺高。穿过。它距地面约 20 英尺，完全 被一堵由扭曲的树枝组成的墙包围着。您可以看到间隔 均匀、窗户大小的间隙，以及最靠近花园平台的位置似 乎是一扇门。 
	
	最后一个最高的平台距地面大约 30 英尺，比其他平 台小得多。它看起来通过另一个小斜坡与中央平台相 连。 
	
	平台下方散布着两间小木屋和一间大户外屋。

当小队到达复合屋的时候，草坪上正有三只要么在睡 觉，要么正在玩一对超大的骰子的 <span style="color:rgb(192, 0, 0)"><b>猿猴</b></span> 。它们身旁的泥 土里倒插着几把铁制巨剑。一头 <span style="color:rgb(192, 0, 0)"><b>棕熊</b></span> 正在外 屋里处理些事情。

中央平台的大门几乎总是锁着的。玩家们需要一个 <span style="color:rgb(255, 192, 0)"><b>DC12 的力量检定</b></span> 来打破锁或者用 <span style="color:rgb(255, 192, 0)"><b>DC12的灵巧检定</b></span> 来把它撬开。里面的区域分为三个部分

1. 一个图书馆和实验室的复用房间
2. 一个客厅和一个房间/厨房。
	杂乱的书架排满了外墙，工作桌上则被墨迹斑驳的笔记和各类野兽怪物的解剖图堆满。
3. 最高处的那层现在是诺克的卧室。
	里面除了衣柜，一个大的木床以及杂乱无章的梳妆台几乎什么也没有。

每个小屋则有两到三张床， 有几张床过大了，不也也有普通人大小的。

**[[阿西穆德·诺克 Ahmed Noke]]** 正在中央平台那里狂热地研究着他的新法术。如果注意到小队靠近，他会说道

	“交出那只绵羊，不然我就把你们撕成碎片”。
	
如果他们开始谈话，诺克将说明他憎恨闪布莱特的原因，并且大肆吹嘘自己的成就。他同时也会问到小队是否杀死了

	“我的好兄弟，古兹”
	
如果确实如此，那么诺克将表现出明显的沮丧。最后，如果玩家们拒绝交出 **[[芬塞尔·闪布莱特 Finethir Shinebright]]** ，那么他会命令手下们开始攻击。战斗的关键在于 **[[阿西穆德·诺克 Ahmed Noke]]** 本人，尽管他的进攻能力相对薄弱，但是却可以非常有效地增强他的盟友。一旦 <span style="color:rgb(192, 0, 0)"><b>棕熊</b></span> 从外屋出现，诺克将对使用 <span style="color:rgb(112, 48, 160)"><b>诅咒</b></span> 来协助。同时他会用 <span style="color:rgb(112, 48, 160)"><b>混沌涌动 </b></span>来辅助盟友的攻击，但是还是专心于维持他的专注上。<span style="color:rgb(192, 0, 0)"><b>猿猴</b></span> 们会有智力地行动，因为挥舞着巨剑，他们的拳头攻击被挥砍所替换，并且造成 10（2d6+3）点物理伤害。<span style="color:rgb(192, 0, 0)"><b>猿猴</b></span> 们不太会死战到底，降到半血以下后他们会逃离战斗。

一旦战局已定，或者单纯因为一个玩家过于靠近，**[[阿西穆德·诺克 Ahmed Noke]]** 将逃进中央平台里，并把身后的门锁上。他会立刻逃进他的房间，并且对他能找到的最大的物体施展 <span style="color:rgb(112, 48, 160)"><b>完全变形术</b></span> ：他的炼金道具。 当 **[[阿西穆德·诺克 Ahmed Noke]]** 逃离战场 3 轮以后，他将骑着 <span style="color:rgb(192, 0, 0)"><b>锻造巨魔</b></span> 从卧室的天花板破洞而出。这巨魔看起来就像人型的炼金道具，火炉组成了它的核心，胸口由导热管组成的炮口。如果玩家们还在客厅，那么它将冲破客厅的墙壁并开始攻击。

这个 <span style="color:rgb(192, 0, 0)"><b>锻造巨魔</b></span> 并不聪明，它会尽量使用自己的 <span style="color:rgb(112, 48, 160)"><b>随机战术</b></span> ，如果当喷吐武器无法使用时，它将单纯攻击自己身边最近的人。尽管如此，它依旧会优先攻击那些用火战斗的角色。当骑在 <span style="color:rgb(192, 0, 0)"><b>锻造巨魔</b></span> 背上时，**[[阿西穆德·诺克 Ahmed Noke]]** 只会使用 <span style="color:rgb(112, 48, 160)"><b>诅咒</b></span> 来减慢任何试图撤退的角色。

如果 <span style="color:rgb(192, 0, 0)"><b>锻造巨魔</b></span> 被武力摧毁或是 **[[阿西穆德·诺克 Ahmed Noke]]** 无法维持自己的专注，它会变回一堆破烂不堪的炼金装置。

在绝望的操控中， **[[阿西穆德·诺克 Ahmed Noke]]** 不愿看到昔日的老师再次获胜，他一边含糊不清地说着什么，一边对自己使用了魔杖，试图变形为某种怪物。然而，被过度使用的魔杖将会裂开，在魔法的滋滋声中，失效的魔杖将会发出一声巨响，将诺克变成一只 <span style="color:rgb(192, 0, 0)"><b>惧噬体</b></span> ——一滩长满怪异的巨口的扭曲血肉。这只可悲的生物会无脑地开始攻击，直到被摧毁为止。
